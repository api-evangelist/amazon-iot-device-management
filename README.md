# Amazon IoT Device Management (amazon-iot-device-management)
AWS IoT Device Management makes it easy to securely onboard, organize, monitor, and remotely manage IoT devices at scale. It enables you to register your connected devices individually or in bulk, and easily push updates to your devices to keep your fleet secure and running smoothly.

## APIs

### AWS IoT Device Management API
The AWS IoT Device Management API provides programmatic access to manage IoT device fleets including thing groups, indexing, jobs, tunneling, and fleet hub for large-scale device management operations.
